Suggestream is a video game recommendation website created by Stuart Purcell, a Computing student at the University of Sunderland.

This site allows you to create an account, browse game genres and receive personalised recommendations.

Each game also features a Twitch stream clip in its detail page, this is to provide you with a genuine representation of the game and hopefully help you decide whether the game is for you.

Notice: this is a work in progress so please report bugs to Stuart in the questionnaire if you see any.
Feedback and future recommendations are also welcomed in order to improve the site.
